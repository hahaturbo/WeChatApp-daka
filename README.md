# 极简打卡小程序（微信）

- 极简打卡，一款极简的打卡小程序
- 没有圈子等复杂的功能，只为专心打卡
- 安安静静的
- 完成自己设定的小目标
- 从此刻开始

## 开发说明

使用node开发

```bash
npm install
```

开发模式

```bash
yarn start # 开发模式打包并热更新
```

```bash
yarn preview # 生产模式打包并热更新
yarn bulid # 生产模式打包
```
